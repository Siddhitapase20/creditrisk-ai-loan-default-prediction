# CreditRisk AI: Loan Default Prediction using Machine Learning

## Overview
CreditRisk AI is an end-to-end Machine Learning project that predicts loan default risk using customer demographic and financial information.

## Features
- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Engineering
- Random Forest Classifier
- XGBoost Classifier
- Cross Validation
- Hyperparameter Tuning
- Model Explainability (SHAP)
- Streamlit Web App

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
- Streamlit
